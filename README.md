# Enactus SHA — Official Website

The official website for the **Enactus** student activity chapter at **El-Shorouk Academy**, built and deployed as a live web application.

🌐 **Live Site**: [enactus-sha.vercel.app](https://enactus-sha.vercel.app)

---

## 🛠️ Built With

- **React** — UI framework
- **TypeScript** — type-safe JavaScript
- **Vite** — fast build tool and dev server
- **Tailwind CSS** — utility-first styling
- **shadcn/ui** — accessible component library
- **Supabase** — backend and database
- **Vercel** — deployment and hosting

---

## 🌍 Deployment

The project is deployed on **Vercel**. Every push to the `main` branch triggers an automatic redeployment.
